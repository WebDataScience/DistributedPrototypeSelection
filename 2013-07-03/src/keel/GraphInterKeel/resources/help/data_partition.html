PARTITION

All the DataSets created by the user or transformed from UCI files, are called "Complete DataSets" and cannot be used in experiments. First it's necessary to divide the whole file in pairs of training and test files.

To divide a file, you must select the "Complete DataSet" and indicate the root name of a directory that will contain all the files generated during the partition process. The name of the files is given automatically and depends on the type of the partition selected. Viewing the name it is very easy to recognize the type of the partition applied.

There are four partition types available:

  • K - Fold Cross Validation.
  • Partitions by defined percentages.
  • 5x2 Cross Validation.
  • Stratification.

Depending the one you select, differents parameters of partitions are shown. You can modify them to customize the partition process.
