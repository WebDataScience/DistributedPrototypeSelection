NEW DATASET

This process can be done in three different stages, according to the tabs at the top zone:

  1. Heading:

    The DataSet's heading contains information about this content: attributes, types, rank of values, etc. All the information is defined from user's interface in a simply way, because user only need to introduce the name of the DataSet or Relation, and then repeat this steps until it's necessary:

    1. Introduce the attribute name.
    2. Select the attribute's type. If the type is "Integer" or "Real" you must indicate the rank of values allowed. However, if the type is "Nominal", you must fill the list with the correct values.
    3. You must specify the attribute function.

    Using the 'Add' button, the attribute defined is added to the DataSet's heading. You can remove an attribute from the heading by using the 'Remove' button.


  2. Data:

    At this screen, you must fill the content of the DataSet. There is two ways for doing it, typeing all the data (data must be separated by comas) or importing a file containing them.


  3. Save:

    If you have defined the heading and filled the content of the DataSet, now you only need to Save the new DataSet in a file. At this stage, the content of the DataSet is validated according to the heading, and if an error is detected, a message is shown.
