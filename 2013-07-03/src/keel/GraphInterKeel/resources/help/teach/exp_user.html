USER'S METHODS

The Keel's experiment design tool allows you to insert your own algorithms whenever they follow the standards on the file input/output formats of. Yours programs can be written in any programming language. This provide a flexible method to compare with the already existing algorithms.

  • Insert user's method
    In order to insert an user's method, press the button User's Method located in the tool bar. Next, click with the left button of the mouse on the draw area.
  • Configure user's method
    Configure algorithm doing a double click on him or through the emergent menu that will appear pressing the mouse's right button (option Show Parameters). The first dialog allows you to introduce the paths of the algorithm and the pattern file. If the user's method needs some specific command to run, write it in the corresponding zone (for example: java - jar, perl, etc...).

    After that you can configure its parameters like any other algorithm although, in this case, in addition, you will have to indicate the type of the algorithm.
